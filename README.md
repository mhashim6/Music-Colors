# Music-Colors
**This is a simple module to change The colors of the Music (Walkman) app. These are the colors that can be changed:**

 - Primary Colors
	 - Action Bar color
	 - Status Bar color
	 - Accent Color
 - Dark Theme
	 - Progressbar Fill
	 - Progressbar Background
 - White Theme
	 - Progressbar Fill
	 - Progressbar Background

> Supported Android Versions: 4.1+
